## CSV Guide

### Folders

| Folder Name           | Description                                         | Purpose                                              |
|-----------------------|-----------------------------------------------------|------------------------------------------------------|
| all-cross-temp        | All valid data, sorted by temperature and then manufacturer | For deriving VOC thresholds with the most possible data        |
| all-cross-temp-high-st| All valid data collected with high ST, sorted by temperature and then manufacturer | For fairly comparing ST and VOC as low battery signals |
| level-li-cross-temp   | Level Li batteries only, sorted by temperature       | To optimize for level li batteries |
| meta-testing          | Tests to define the optimal test parameters          | Determine best testing conditions                    |
